# В чём суть задания?
Есть ==алгоритм==, на вход которого подаётся ==натуральное число==, которое обрабатывается в несколько шагов определённым способом, описанным в задании:
	*1*. Строится новая запись числа, путём перевода его в какую-либо **систему счисления**.
	2. Далее, с этой записью числа ==производятся действия==, могут быть **разделения на условия**(*например, если полученная запись кратна трём*)
	3. Полученное число ==переводится в **десятичную систему счисления**==.
Используя данный алгоритм, нужно определить какое **число**(или какое **минимальное число**) было передано в алгоритм, если **получившееся число** равно определённому значению либо как-то соотносится к этому значению(*больше, либо меньше*).
# Что нужно понимать для задания?
==Данная задача решается в Python==.
1. Нужно знать, как **осуществляется** перевод в любую систему счисления и как написать для этого функцию. Если число переводится в *двоичную, восьмеричную и шестнадцатеричную* систему счисления, можно ==воспользоваться функциями== **`bin(a)[2:],  oct(a)[2:], hex(a)[2:]`** соответственно, где **a** - число, а полученный результат - **строка**. 
2. **Обязательно** нужно знать функцию **`int(a,n)`**, где **a** - число в виде строки, **n** - система счисления числа **a**. Данная функция возвращает ==число **a** в представлении десятеричной системы счисления==.
3. Так же следует понимать, как работать с циклом **`for`**, чтобы найти ==минимально возможное значение== числа, которое поступает на вход в алгоритм.
4. Почти все задания включают в себя **работу со строками**(*посчитать количество единиц, записать в начало/конец какое-либо число и так далее*), поэтому так-же нужно знать ==методы строк== и как можно ==изменять/получать значение по индексу== в строке(*важно помнить, что счёт всегда идёт с нуля*). 
# Как выполнять задание?
## Через код:
##### Первый шаг:
Написать функцию ==перевода числа в данную систему счисления==, обязательно использовать для формирования числа **строку**.
Принцип перевода числа в любую систему счисления до 20:
	*1*. Если число равно нулю, ==число таким и остаётся==
	2. Пока число больше нуля, ==число делится на основание системы счисления==.
	Данный шаг повторяется, пока число **можно** делить на основание системы счисления.
	*3*. Все остатки от деления ==записываются в обратном порядке==, это и будет являться новым полученным числом. *Например, если в процессе деления были получены остатки 1,0,0,1,1, то полученное число будет 11001*. 
	Если остаток от деления **больше 9**, оно заменяется следующим буквенным обозначением:
		*10 - A
		11 - B
		12 - C
		13 - D
		14 - E
		15 - F
		16 - G
		17 - H
		18 - I
		19 - J*
Важно помнить, что любая система счисления ==начинается с нуля==. 
##### Второй шаг:
Используя цикл **`for`**(*если он нужен*), написать алгоритм, данный в задаче. Для удобства, переменную числа и переменную новой записи числа можно назвать **N** и **R**, как это, обычно, пишут в задачах. 
==Внутри блока цикла==, сначала инициализируется переменная **R**, равная числу **N** в заданной системе счисления. Далее, с этим числом ==производятся действия==. Они могут зависеть от условия, данного в тексте, поэтому нужно воспользоваться условными операторами **`if`** и **`else`**. Полученное число переводится ==обратно в десятеричную систему счисления==.
##### Третий шаг:
Конечный результат ==нужен для выполнения задания==(*например, проверить, если оно больше 17*). Проверив все условия, запустить код и записать в ответ число **N, которое было дано в алгоритм**.
### Шаблон на Python:
Так как второй пункт алгоритма совершенно разный в каждом задании, **шаблон не является автокодом**.
```
def f(n):  # функция перевода в систему счисления
	s = ''
	if n == 0:
		return '0'
	while n > 0:
		s = str(n%[основание_сис_счисления]) + s
		n = n // [основание_сис_счисления]
	return s

# Если задача требует найти минимальное число N, при котором выполняется какое-либо условие, нужно использовать цикл for.
for N in range(1000): # 1000 - произвольное значение, чтобы точно поймать нужное значение.
	R = f(N) # важно помнить, что получившееся значение - это строка.
	# первый пункт алгоритма написан, дальше требуется совершить совершенно случайные действия со строкой. Допустим, требуется добавить в конец строки 0, если первая цифра числа делится на 3, или добавить в начало строки 1, если первая цифра числа не делится на 3
	if R[0]%3 == 0:
		R = R + '0'
	else:
		R = '1' + R
	# второй пункт алгоритма написан, дальше нужно перевести число обратно в десятичную систему счисления
	R = int(R,[текущая_система_счисления_числа])
	# допустим, дальше требуется найти минимальное число N, при котором R будет больше 20
	if R > 20:
		print(N)
		break
# при запуске кода на экране высветится минимальное число N, при котором R будет больше 20
```
