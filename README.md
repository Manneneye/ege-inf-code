## Что из себя представляет?
В данном репозитории находятся решения различных задач **№2,5,6,8,11,13,14,15,16,17,19,20,21,23,24,25,26,27** из ЕГЭ по информатике 2026 года при помощи языка программирования Python. Используются **ТОЛЬКО** встроенные библиотеки, такие как: itertools, functools, turtle, и т.д.
Все задачи берутся из Яндекс Тренажёра(включает в себя авторские задачи различных преподавателей и задачи из ФИПИ)
## Как работать с этим?
Репозиторий разбит на папки, названия которых соответствуют номеру задания. Внутри находятся файлы ```.py```, содержащие в себе описание и решение задачи. Там же находится небольшой ликбез о самом задании, что в нём требуют, как решать, как выглядит и тому подобное. Файлы .md писались в программе Obsidian, применение которой очень рекомендуется для работы с этими заметками
