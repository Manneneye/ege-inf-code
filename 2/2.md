# В чём суть задания?
Дана таблица истинности ==*логической функции F*==, которую заполнял какой-то человек. Этот человек успел заполнить лишь фрагмент таблицы из нескольких её различных строк(то есть ==**некоторая часть значений пропущена**==), даже не указав, какому столбцу таблицы соответствует каждая из переменных(это говорит о том, что ==переменные *могут идти не в том порядке, в каком их пишут*==). Однако значение функции для каждой строки записано и оно всегда одинаково. Сутью задания является ==соотнесение== к каждому столбцу таблицы какую-либо переменную.
# Логическая функция
Под *логической функцией* в данном задании понимается функция, содержащая в себе ==логические операторы и переменные==(как правило, их 4). Переменные могут принимать значение *0* или *1*, следовательно, *функция тоже может принимать значение 0 или 1*. 
##### Логические операторы:
Описание операторов разделится на 3 пункта: *смысл* оператора(при каких значениях переменных принимает 0 либо 1), *обозначение* и *пример*.
- **Инверсия(*отрицание*)**: 
	*Смысл*: заменяет 0 на 1 или наоборот. 
	*Обозначение*: `¬x`
	*Пример*: `¬1 = 0`,  `¬0 = 1`
- **Конъюнкция(*логическое умножение*)**: 
	*Смысл*: выражение будет равно 1, если оба значения переменных равны 1. В остальных случаях выражение будет равно 0. 
	*Обозначение*: `x ∧ y`
	*Пример*: `1 ∧ 0 = 0` ,  `0 ∧ 0 = 0`,  `1 ∧ 1 = 1`  
- **Дизъюнкция(*логическое или*)**:
	*Смысл*: выражение будет равно 1, если хотя бы одно из значений переменных равно 1. Если оба значения переменных равно 0, то и выражение будет равно 0.
	*Обозначение*: `x ∨ y`
	*Пример*: `1 ∨ 0 = 1`, `0 ∨ 1 = 1`, `0 v 0 = 0`
- **Импликация(логическое следование)**:
	*Смысл*: если из переменной, равной 1, следует переменная, равная 0, то выражение будет равно 0. В остальных случаях выражение будет равно 1.
	*Обозначение*: `x ⇒ y`
	*Пример*: ==`1 ⇒ 0 = 0`==, `0 ⇒ 1 = 1`, `0 ⇒ 0 = 1`, `1 ⇒ 1 = 1`
- **Эквиваленция(равенство/тождественность)**:
	*Смысл*: выражение будет равно 1, если переменные равны между собой. В противном же случае, выражение будет равно 0
	*Обозначение*: `x ≡ y`
	*Пример*: `1 ≡ 1 = 1`,`0 ≡ 1 = 0`,`1 ≡ 0 = 0`, `0 ≡ 0 = 1`
# Как выполнять задание?
## Через код:
##### Первый шаг:
Вывести на экран все переменные ==через пробел==.
##### Второй шаг:
Затем записать вложенные циклы *for* ==по количеству переменных== в радиусе ==0 до 1==, объявляя соответствующую переменную
##### Третий шаг:
Внутри блока последнего цикла записать **условный оператор**, в котором *сравнивается ==логическое выражение== с переменными вложенных циклов ==со значением в таблице==*. Если это условие **правдивое**, вывести на экран значения переменных. 
##### Четвёртый шаг:
Запустить код и ==сопоставить== значения из полученной таблицы с данной таблицей и записать в ответ названия переменных **в таком порядке, в котором они расположены**.
